Antichamber

Gets one random sentence of life advice from the game Antichamber.

./antichamber

Useless Web

Gets one random useless website url. Websites from The Useless Web.

./uselessweb

Hyperlinked

Gets a list of Hyperlinked characters that match the attribute info provided.

./hyperlinked/{attribute}/{data}

attribute: Type of attribute. ("name", "age", "size", "gender")

data: Piece of information.

name: String (case-insensitive) that represents the full name. ("spotify", "4Chan", "YouTube")

age: Integer that represents the age in years.

size: Integer that represents the height in centimeters.

gender: String that represents the gender. ("male", "female", "unknown" for 4chan)

I could add sexualities if I can find a reliable source
